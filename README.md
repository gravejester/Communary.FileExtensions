#Communary.FileExtensions

A PowerShell module with functions to enhance and simplify the way you work with files and folders.

##Functions
- Invoke-FastFind*
- Get-DirectoryInfo*
- Remove-File*
- New-TempFile
- Get-TempFolder
- Get-RandomFileName
- Resolve-PathEx
- Invoke-Touch
- Set-FileAttribute*
- Get-FileAttributes*

*These functions utilize the Win32 API to give support for long paths.
